Заголовок: Метод contains возвращает True при поиске пустого символа в непустой строке
Серьезность: Критический
Приоритет: Средний
Шаги воспроизведения:
1. Создать экземпляр класса StringUtils
2. Вызвать метод contains с параметрами:
    - string = "empty" (непустая строка)
    - symbol = "" (пустая строка)
3. Получить результат
Ожидаемый результат: False (пустой символ не находится в непустой строке)
Фактический результат: True (в непустой строке находится пустой символ)
Окружение: 
    - Устройство: Ноутбук Lenovo IdeaPad Gaming 3 15IMH05, 15.6"
    - ОС: Windows 11, Python 3.14.2, pytest 9.0.2
